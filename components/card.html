# Card 卡片

通用的卡片容器组件,支持标题、额外操作区域、阴影效果等。

# 基础用法

包含标题和内容的基础卡片。

用户信息

这是卡片的内容区域

可以放置任何内容

<template>
  <z-card title="用户信息">
    <p>这是卡片的内容区域</p>
    <p>可以放置任何内容</p>
  </z-card>
</template>

# 带操作区域

通过 extra 属性或插槽设置卡片右上角的操作区域。

用户列表

张三 - 管理员

李四 - 编辑

王五 - 访客

<template>
  <z-card title="用户列表">
    <template #extra>
      <el-button type="text" @click="handleMore">查看更多</el-button>
    </template>
    <div>
      <p>张三 - 管理员</p>
      <p>李四 - 编辑</p>
      <p>王五 - 访客</p>
    </div>
  </z-card>
</template>

<script>
export default {
  methods: {
    handleMore() {
      this.$message.info('查看更多用户')
    }
  }
}
</script>

# 不同阴影效果

通过 shadow 属性设置卡片的阴影显示时机。

总是显示

shadow = always

卡片总是显示阴影

鼠标悬浮

shadow = hover

鼠标悬浮时显示阴影

从不显示

shadow = never

卡片从不显示阴影

<template>
  <div>
    <el-row :gutter="20">
      <el-col :span="8">
        <z-card title="总是显示" shadow="always">
          <p>shadow = always</p>
          <p>卡片总是显示阴影</p>
        </z-card>
      </el-col>
      <el-col :span="8">
        <z-card title="鼠标悬浮" shadow="hover">
          <p>shadow = hover</p>
          <p>鼠标悬浮时显示阴影</p>
        </z-card>
      </el-col>
      <el-col :span="8">
        <z-card title="从不显示" shadow="never">
          <p>shadow = never</p>
          <p>卡片从不显示阴影</p>
        </z-card>
      </el-col>
    </el-row>
  </div>
</template>

# Props

参数 说明 类型 默认值
title 卡片标题 String -
extra 卡片右上角的操作区域 String -
shadow 阴影显示时机 (always/hover/never) String 'always'
body-style 内容区域的自定义样式 Object -
header-style 标题区域的自定义样式 Object -

# Slots

插槽名 说明
default 卡片内容
header 自定义标题区域内容
extra 卡片右上角的操作区域
